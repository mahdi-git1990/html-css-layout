# html-css-layout
A responsive page built html - css

# طراحی ساده با HTML و CSS

این پروژه یک طراحی ساده و کامل رابط کاربری با استفاده از HTML خالص و CSS است.

## هدف پروژه
پیاده‌سازی یک صفحه با ساختار بصری ساده و ریسپانسیو، با تمرکز بر استفاده صحیح از تگ‌های HTML و اصول CSS

## تکنولوژی‌های استفاده شده
- HTML5
- CSS3

## ساختار فایل‌ها
- `index.html`: فایل اصلی صفحه.
- `assets/`: پوشه‌ای شامل:
    - `css/`: فایل‌های CSS 
    - `fonts/`: فونت‌های استفاده شده 
    - `js/`: کد های js 
    - `images/`: تصاویر استفاده شده در پروژه
